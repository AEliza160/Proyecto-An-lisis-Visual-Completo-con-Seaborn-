# Proyecto-An-lisis-Visual-Completo-con-Seaborn-
Dashboard Climático 2020–2025

Este proyecto analiza datos relacionados con el cambio climático global entre 2020 y 2025.
El objetivo es explorar las variables, visualizarlas y crear un pequeño dashboard que muestre las tendencias más importantes.

 1. Contenido del Proyecto

El notebook incluye:

Carga del dataset

Revisión de dimensiones, tipos de datos y nulos

Visualizaciones individuales

Matriz de correlación

Dashboard con subplots

Guardado de gráficas en alta calidad

Insights finales del análisis

2. Dataset Utilizado

El dataset contiene información sobre diferentes variables climáticas entre 2020 y 2025.
Se trabajó con datos limpios (sin valores nulos).

3. Visualizaciones

Se realizaron varias gráficas para analizar:

Temperatura promedio

Emisiones

Aumento del nivel del mar

Contaminación

Precipitaciones

Otras variables relacionadas

Además, se creó:

 una matriz de correlación

 un dashboard con subplots donde se muestran todas las visualizaciones juntas

 4. Resultados (Insights)

Al final se presentan conclusiones simples basadas en los gráficos generados, como tendencias, aumentos, bajadas o relaciones entre variables.

5. Tecnologías Utilizadas

Python

Pandas

Matplotlib

Seaborn

Google Colab

6. Cómo Ejecutarlo

Descargar o clonar el repositorio
